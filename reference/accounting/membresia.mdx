## Overview

`Membresia` encodes a student's subscription to the academy. Two creation paths exist: a free-form personalized plan and a set of three pre-defined plans with fixed pricing. In both cases, the membership generates revenue (`impactoComercial()` returns the price) that is added to the academy's capital by the controller.

```java
package com.nousacademy.contabilidad;

public class Membresia implements ComportamientoContable, Serializable { ... }
```

## TipoMembresia

```java
public enum TipoMembresia {
    BASICA, INTENSIVA, PREMIUM, PERSONALIZADA
}
```

### Plan pricing (per month)

| Plan | Monthly price (COP) | Private classes per month |
|------|--------------------|--------------------------|
| `BASICA` | 1,200,000 | 0 |
| `INTENSIVA` | 2,200,000 | 4 |
| `PREMIUM` | 3,500,000 | 8 |
| `PERSONALIZADA` | Set by administrator | Set by administrator |

<Note>
For pre-defined plans, the total `precio` is `pricePerMonth × meses` and `clasesPrivadas` is `creditsPerMonth × meses`.
</Note>

## ComportamientoContable interface

`Membresia` implements `com.nousacademy.contabilidad.ComportamientoContable`.

### `impactoComercial()`

```java
@Override
public double impactoComercial()
```

Returns `this.precio`. The controller adds this value to `capital` when assigning the membership.

**Return:** the total membership price (positive value — income for the academy).

### `generarCodigo(String sufijo)`

```java
@Override
public String generarCodigo(String sufijo)
```

Increments the static serial and returns `sufijo + String.format("%04d", serial)`. Always called with `"MEM"`, producing codes like `MEM0001`, `MEM0002`, …
