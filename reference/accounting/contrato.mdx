title: Contrato

## Overview

`Contrato` encodes a time-bounded employment agreement between the academy and a teacher. The contract generates an expense — `impactoComercial()` returns `-salarioBase × meses` — which is subtracted from academy capital by the controller when the contract is created.

```java
package com.nousacademy.contabilidad;

public class Contrato implements ComportamientoContable, Serializable { ... }
```

## EstadoContrato

```java
public enum EstadoContrato {
    ACTIVO, SUSPENDIDO, FINALIZADO
}
```

| State | Meaning |
|-------|---------|
| `ACTIVO` | Contract is active. Initial state. |
| `SUSPENDIDO` | Contract is temporarily suspended. |
| `FINALIZADO` | Contract has ended. The `Docente` reference is set to `null` when this transition is made via `actualizarEstadoContrato`. |

## ComportamientoContable interface

`Contrato` implements `com.nousacademy.contabilidad.ComportamientoContable`.

### `impactoComercial()`

```java
@Override
public double impactoComercial()
```

Returns `-this.salarioBase * this.meses`. This negative value is added to `capital` in the controller, effectively reducing it.

**Return:** negative double representing total contract cost.

### `generarCodigo(String sufijo)`

```java
@Override
public String generarCodigo(String sufijo)
```

Increments the static serial and returns `sufijo + String.format("%03d", serial)`. Always called with `"CON"`, producing codes like `CON001`, `CON002`, …
