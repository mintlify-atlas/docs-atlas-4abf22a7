## Overview

A `Reserva` combines a `Clase` (either `ClaseGrupal` or `ClasePersonalizada`) with real-time scheduling data, participant lists, and lifecycle state. It is created through `AcademiaController` and persisted to `datos/<codigo>.res`.

```java
package com.nousacademy.modelo;

public class Reserva implements Serializable { ... }
```

## EstadoReserva

`EstadoReserva` is an enum in `com.nousacademy.modelo` that represents every possible lifecycle state.

| Value | Meaning |
|-------|---------|
| `PENDIENTE` | Created, waiting for students and a teacher. Initial state. |
| `LLENA` | Enrollment has reached `aforoMaximo` (or a private class has one student). |
| `EN_CURSO` | Class has been started by the teacher. |
| `EXITOSA` | Class ended normally (real duration ‚â• expected ‚àí 25 min). |
| `PREMATURA` | Class ended more than 25 minutes early. |
| `TARDE` | Class was started more than 10 minutes past schedule. |
| `CANCELADA` | Class was cancelled by the student (private) or administratively. |
| `SANCIONADA` | Reserved for future sanction logic. |

### State transitions

```
PENDIENTE ‚îÄ‚îÄenroll student‚îÄ‚îÄ‚ñ∫ LLENA
PENDIENTE / LLENA ‚îÄ‚îÄiniciarClase (on time)‚îÄ‚îÄ‚ñ∫ EN_CURSO
PENDIENTE / LLENA ‚îÄ‚îÄiniciarClase (> 10 min late)‚îÄ‚îÄ‚ñ∫ TARDE  ‚Üí  DocenteRetardado thrown
EN_CURSO ‚îÄ‚îÄfinalizarClase (normal)‚îÄ‚îÄ‚ñ∫ EXITOSA
EN_CURSO ‚îÄ‚îÄfinalizarClase (> 25 min early)‚îÄ‚îÄ‚ñ∫ PREMATURA  ‚Üí  FinalizacionPrematuraDeReserva thrown
* ‚îÄ‚îÄcancelacionAdministrativa()‚îÄ‚îÄ‚ñ∫ CANCELADA
ClasePersonalizada ‚îÄ‚îÄcancelarReserva(student)‚îÄ‚îÄ‚ñ∫ CANCELADA
```

<Note>
Rating is allowed in states `EXITOSA`, `PREMATURA`, and `TARDE`. Joining as an attendee (`unirseClase`) is only allowed in `EN_CURSO`.
</Note>

## Public methods

### `getCodigo()`

```java
public String getCodigo()
```

Convenience method that delegates to `this.clase.getCodigo()`. Returns the class code (e.g., `PB-001`).

### `iniciarClase()`

```java
public void iniciarClase()
    throws DocenteRetardado, ClaseEnEstadoNoIniciar, NoSuficienteAforo
```

Starts the class. Called internally by `AcademiaController.inciarReserva()`.

- Allowed window: 5 min before to 10 min after `horaInicioEsperada`.
- If called after 10 min: sets state to `TARDE`, throws `DocenteRetardado`.
- If called before 5 min: throws `ClaseEnEstadoNoIniciar`.
- For group classes: throws `NoSuficienteAforo` if enrolled count < `aforoMinimo`.
- On success: records `horaInicioReal`, sets state to `EN_CURSO`.

### `finalizarClase()`

```java
public void finalizarClase()
    throws ClaseNuncaIniciada, FinalizacionPrematuraDeReserva
```

Ends the class. Called internally by `AcademiaController.finalizarReserva()`.

- Requires state `EN_CURSO`; otherwise throws `ClaseNuncaIniciada`.
- Records `horaCierreReal` and computes `duracionReal`.
- If `duracionReal < clase.duracion ‚àí 25`: sets state to `PREMATURA`, frees students, throws `FinalizacionPrematuraDeReserva`.
- Otherwise: sets state to `EXITOSA` (or retains `TARDE` if started late), frees students, and calls `docente.trabajoDocente(this)` for `EXITOSA` classes.

### `cancelacionAdministrativa()`

```java
public void cancelacionAdministrativa()
```

Sets state to `CANCELADA` and calls `claseFinalizada()` on every enrolled student. Called by `AcademiaController.eliminarReserva()`.

### `reservarEstudiante(Estudiante estudiante)`

```java
public void reservarEstudiante(Estudiante estudiante)
    throws ClasesPrivadasNoSuficientes, ClaseYaReservada, ClaseLlena
```

Enrols a student. Delegates to `estudiante.reservarClase(clase)` for membership and duplicate checks. Advances state to `LLENA` when the class reaches capacity.

| Throws | Condition |
|--------|-----------|
| `ClaseLlena` | Reservation is already at capacity. |
| `ClaseYaReservada` | Student already has an active reservation. |
| `ClasesPrivadasNoSuficientes` | Personalized class but student has no credits. |

### `asignarDocente(Docente docente)`

```java
public void asignarDocente(Docente docente)
```

Sets the teacher, adds this reservation to the teacher's history, and sets `estadoDocente` to `ACEPTADA`.

### `cancelarDocente()`

```java
public void cancelarDocente()
    throws ReservaNoEnDocente, NingunDocenteAsignado
```

Removes the teacher assignment. Sets `estadoDocente` back to `EN_ESPERA`.

| Throws | Condition |
|--------|-----------|
| `NingunDocenteAsignado` | No teacher is currently assigned. |
| `ReservaNoEnDocente` | The reservation is not in the teacher's history. |

### `cancelarReserva(Estudiante estudiante)`

```java
public void cancelarReserva(Estudiante estudiante)
    throws EstudianteNoEnlistadoEnReserva, ClaseEnEstadoNoCancelar, ClaseNoEnEstudiante
```

Removes the student from `estudiantes` and (if present) `asistentes`. For `ClasePersonalizada`, sets state to `CANCELADA`.

| Throws | Condition |
|--------|-----------|
| `EstudianteNoEnlistadoEnReserva` | Student not in enrolled list. |
| `ClaseEnEstadoNoCancelar` | Current time is within 30 minutes of `horaInicioEsperada`. |
| `ClaseNoEnEstudiante` | Class not in student's history. |

### `unirseClase(Estudiante estudiante)`

```java
public void unirseClase(Estudiante estudiante)
    throws ClaseEnEstadoNoUnirse, EstudianteNoEnlistadoEnReserva
```

Adds the student to `asistentes`. Student must be enrolled and the class must be `EN_CURSO`.

### `calificarClase(Estudiante estudiante, float calificacion)`

```java
public void calificarClase(Estudiante estudiante, float calificacion)
    throws ClaseEnEstadoNOCalificar, EstudianteNoEnlistadoEnReserva, CalificacionInvalida
```

Appends a rating to `calificacionIndividual`, recalculates `calificacionGeneral`, and calls `docente.calificacionDocente()` to update the teacher's aggregate rating.

| Throws | Condition |
|--------|-----------|
| `CalificacionInvalida` | Rating outside 1‚Äì5. |
| `EstudianteNoEnlistadoEnReserva` | Student not in attendees list. |
| `ClaseEnEstadoNOCalificar` | State is not `EXITOSA`, `PREMATURA`, or `TARDE`. |

### `actualizarCalificacionGeneral()`

```java
public void actualizarCalificacionGeneral()
```

Recomputes `calificacionGeneral` as the arithmetic mean of `calificacionIndividual`.

### `actualizarEstado(EstadoReserva estado)`

```java
public void actualizarEstado(EstadoReserva estado)
```

Directly sets the reservation state.

### `actualizarEstadoDocente(EstadoDocente estado)`

```java
public void actualizarEstadoDocente(EstadoDocente estado)
```
