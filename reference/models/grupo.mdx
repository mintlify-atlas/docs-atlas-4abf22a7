title: Grupo
description: Represents a study group that aggregates students, teachers, and reservations under a single code. Students may belong to at most 4 groups.

## Overview

`Grupo` is a serializable container that logically links a set of students, teachers, and reservations. Groups are created by administrators and joined voluntarily by students.

```java
package com.nousacademy.modelo;

public class Grupo implements Serializable { ... }
```

---

## Fields

<ResponseField name="codigo" type="String">
  Auto-generated unique code in `GRUP001` format. Generated by a static serial counter.

<ResponseField name="nombre" type="String">
  Human-readable name of the group.
</ResponseField>

<ResponseField name="descripcion" type="String">
  Description of the group's purpose or topic.
</ResponseField>

<ResponseField name="estudiantes" type="Estudiante[]">
  Array of students currently enrolled in the group. Starts empty.
</ResponseField>

<ResponseField name="docentes" type="Docente[]">
  Array of teachers assigned to the group. Starts empty.
</ResponseField>

<ResponseField name="reservas" type="Reserva[]">
  Array of reservations associated with the group. Starts empty.
</ResponseField>

## Code generation

### `generarCodigo(String sufijo)`

```java
public String generarCodigo(String sufijo)
```

Increments the class-level static counter `serial` and returns `sufijo + String.format("%03d", serial)`.

Always called with `"GRUP"` from the constructor, producing codes like:

```
GRUP001, GRUP002, GRUP003, …
```

<Note>
The counter is static — it is shared across all `Grupo` instances and never resets during an application session.
</Note>

## Business methods

### `anadirDocente(Docente docente)`

```java
public void anadirDocente(Docente docente)
```

Appends the teacher to the `docentes` array. No duplicate check is performed here.

### `anadirEstudiante(Estudiante estudiante)`

```java
public void anadirEstudiante(Estudiante estudiante)
    throws CantidadGruposExcedida, Duplicados
```

Enrols the student in the group. Checks for duplicates using object identity, then calls `estudiante.addGrupo(this)` which enforces the maximum of **4 groups** per student.

| Throws | Condition |
|--------|-----------|
| `Duplicados` | Student is already in this group. |
| `CantidadGruposExcedida` | Student is already in 4 groups (thrown from `Estudiante.addGrupo`). |

### `anadirReserva(Reserva reserva)`

```java
public void anadirReserva(Reserva reserva)
```

Appends the reservation to the `reservas` array.

### `eliminarReserva(String codigoReserva)`

```java
public void eliminarReserva(String codigoReserva)
    throws ReservaNoEncontradaEnGrupo
```

Removes the reservation identified by `codigoReserva` (matched via `Reserva.getCodigo()`). Does not delete the reservation from the controller.

<ParamField path="codigoReserva" type="String" required>Class code of the reservation to remove.</ParamField>

| Throws | Condition |
|--------|-----------|
| `ReservaNoEncontradaEnGrupo` | No reservation with that code is linked to this group. |

### `eliminarEstudiante(Estudiante estudiante)`

```java
public void eliminarEstudiante(Estudiante estudiante)
    throws EstudianteNoEnGrupo, GrupoNoEncontrado
```

Removes the student from the group and calls `estudiante.eliminarGrupo(this)` to keep the student's group list consistent.

| Throws | Condition |
|--------|-----------|
| `EstudianteNoEnGrupo` | Student is not in this group. |
| `GrupoNoEncontrado` | Group not found in the student's list (propagated from `Estudiante.eliminarGrupo`). |

### `eliminarDocente(Docente docente)`

```java
public void eliminarDocente(Docente docente) throws DocenteNoEnGrupo
```

Removes the teacher from the group's `docentes` array using object identity.

| Throws | Condition |
|--------|-----------|
| `DocenteNoEnGrupo` | Teacher is not in this group. |

### `eliminacionAdministrativa()`

```java
public void eliminacionAdministrativa() throws GrupoNoEncontrado
```

Called when an administrator deletes a group. Iterates over `estudiantes` and calls `estudiante.eliminarGrupo(this)` for each, removing the group reference from every enrolled student.

<Warning>
This method is called by `AcademiaController.eliminarGrupo()` before the group object is removed from the controller's array. Do not call it directly.
</Warning>

## Persistence

### `copiarFicheroGrupo(String dir)`

```java
public void copiarFicheroGrupo(String dir) throws IOException
```

Serializes the entire `Grupo` object to the specified path. Called after every mutation to keep `datos/<codigo>.grp` current.

<ParamField path="dir" type="String" required>
  Destination file path, typically `datos/<codigo>.grp`.
</ParamField>

## Getters and setters

| Method | Type | Description |
|--------|------|-------------|
| `getCodigo()` | `String` | Returns the group code (e.g., `GRUP001`). |
| `getNombre()` | `String` | Returns the group name. |
| `setNombre(String)` | `void` | Updates the group name. |
| `getDescripcion()` | `String` | Returns the description. |
| `setDescripcion(String)` | `void` | Updates the description. |
| `getEstudiantes()` | `Estudiante[]` | Returns a clone of the students array. |
| `getDocentes()` | `Docente[]` | Returns a clone of the teachers array. |
| `getReservas()` | `Reserva[]` | Returns a clone of the reservations array. |

<Note>
`getEstudiantes()`, `getDocentes()`, and `getReservas()` return an empty array rather than `null` when the field is uninitialized.
</Note>
